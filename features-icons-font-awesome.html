Font Awesome

You will find intergarted with your template Font Awesome, an pictographic language of web-related actions which delivers over 300 icons. The Font Awesome webfont and CSS files are licensed under CC BY 3.0 and you can find the full examples of usage at Font Awesome - http://fortawesome.github.com/Font-Awesome

Add class fa-CLASS_NAME to any element, best used with a <span>.
